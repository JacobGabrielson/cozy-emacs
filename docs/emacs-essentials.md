# Buffers and Windows

Key commands in Emacs to make sure you never get stuck.

- `C-x C-f` — find/open a file (which really means creating a "buffer" for the file)
- `C-x b` — switch to another buffer by name (note not all buffers are
  for files, some are for shells etc)
- `C-x C-b` — list buffers and pick one
- `C-x o` — switch to the other window
- `C-x 0` — close the current window (buffer stays open; switch back with `C-x b`)
- `C-x 1` — make the current window the only one (hides everything else)
- `C-x 2` — split the current window into two stacked windows
- `C-x 3` — split the current window into two side-by-side windows
- `C-x 4 f` — find/open a file in the other window
